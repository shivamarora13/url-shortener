API Basejump: URL Shortener Microservice

This is a url shortening microservice. The purpose of this microservice is to be able to parse a url in the format of `http://www.google.com` or `https://www.google.com` to the `/new` endpiont, assign that url to an internal number, and redirect the user to that url when they visit `https://cryptic-reef-58313.herokuapp.com/int`.

Example Creation Usage

https://cryptic-reef-58313.herokuapp.com/new/http://www.google.com

Example Creation Output

{"original_url": "http://www.google.com","short_url": "https://cryptic-reef-58313.herokuapp.com/1"}

Example Usage

`https://cryptic-reef-58313.herokuapp.com/1` will redirect to `www.google.com`.
